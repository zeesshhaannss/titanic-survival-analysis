# Titanic Survival Analysis

## Overview
Predicting passenger survival on the Titanic using machine learning.

## Dataset
- 891 passengers with 17 features (pre-processed)
- Target: Survived (0 = No, 1 = Yes)

## Analysis Performed
1. **Exploratory Data Analysis** - Distribution analysis, survival rates by demographics
2. **Statistical Testing** - Correlation analysis, significance tests
3. **Predictive Modeling** - Logistic Regression model
4. **Results** - 80%+ test accuracy

## Key Findings
- Women had 74% survival rate vs 19% for men
- 1st class passengers had highest survival
- Fare/wealth was strong predictor
- Family size impacted survival chances

## Tools Used
- Python, Pandas, NumPy
- Scikit-learn (Logistic Regression)
- Matplotlib, Seaborn (Visualization)
- Jupyter Notebook

## Model Performance
- Training Accuracy: 81%+
- Test Accuracy: 80%+
- Confusion Matrix included
